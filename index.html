Plan C の長期コストリスク： 20年ベースでは Plan C が最安（2.33億円）だが、現図書館は築45年（1981年建設）。改修から20年後には築65年となり、 躯体の耐用年数・設備更新コストが再度発生するリスクが高い。 超長期ではPlan C のコスト優位性が低下する可能性がある。
⚠️ Plan C 選択時の隠れリスク
Plan C（現図書館改修）を選択した場合でも、ピュアモルトクラブハウスの空調整備（約4.52億円）は別途・将来的に必要となる。 この費用は Plan C の事業費には含まれておらず、実質的な町の財政負担は試算額を上回る可能性がある。 Plan C の「低コスト」評価は、この潜在的支出を除外した数字である点に留意が必要。
※修繕費は公共施設の長寿命化計画モデルに基づく建設費（再調達価額）の25%で試算（ライフサイクルコスト.md・草津市将来コストシミュレーション等参照）。 実質持出＝補助金・交付税措置控除後の町の純負担。Plan B の建物修繕費は貸主負担のため町負担なし（ただしテナント契約条件に依存）。 Plan C の試算は昇降設備等の追加工事費を含まない現状事業費ベース。

⚠ 追加検討事項（面積・蔵書数の試算）

テナント450㎡（開架30,000冊）に別棟閉架書庫（80㎡・20,000冊）を加えて合計50,000冊規模を確保する。現有86,000冊から36,000冊の縮小となるが、閉架書庫の別棟整備により収蔵機能を補完する。答申書が求める蔵書規模との整合性確保が引き続き課題となる。

蔵書数の大幅減少を補うために閲覧席・読書環境・空間デザインに格別の工夫が必要である。また、蔵書を補完するために別棟として閉架書庫を建築する選択肢もあるが、その場合は事業規模がさらに拡大する。

商業施設のテナントとして図書館を位置づけるためには、テナント料（1,600万円/年・30年で約4億8,000万円）に見合う「図書館」としての機能的・文化的効果を明確なストーリーで示す必要がある。

財源解説公共施設等適正管理推進事業債

公共施設等適正管理推進事業債（以下「公管推進事業債」）は、地方公共団体が公共施設等の適正管理を行うための事業に充当できる地方債。充当率90%、事業の種別に応じて元利償還金の一定割合が普通交付税措置される。

対象事業の区分と交付税措置

① 集約化・複合化事業
交付税措置：50%
複数施設を集約・複合化する事業。充当率90%。面積削減が条件。
② 転用事業
交付税措置：40%
既存施設を用途転換する事業。充当率90%。
③ 除却事業
交付税措置：50%
（単独除却は措置なし）
老朽化施設の解体除却。集約化等に伴う除却は50%措置。単独除却は措置なし。
④ 長寿命化事業
交付税措置：30%
長寿命化計画に基づく改修。充当率90%。

12-1. 図書館法第2条（定義）と各プランの設置形態

図書館法第2条は図書館を「図書、記録その他必要な資料を収集し、整理し、保存して、一般公衆の利用に供し、その教養、調査研究、レクリエーション等に資することを目的とする施設」と定義する。設置者は地方公共団体・日本赤十字社・一般社団／財団法人に限定され、地方公共団体設置のものを公立図書館という。

Plan A・B・Cはいずれも鹿追町（地方公共団体）が設置する公立図書館として位置づけられる。

用語解説主要用語の定義

用語 定義
リノベーション

既存の公共建築に大規模な改修を行い、建物の機能・性能・空間構成などを抜本的に見直すことで、従前より高い価値や新たな用途を創出する改修。

耐震性・断熱性の向上、バリアフリー化、設備更新と併せたレイアウト変更等により、利用者ニーズや時代の要請に応じた公共サービス水準の向上を目的とする。
リフォーム
（改修）

既存の公共建築について、間取りや仕上げ、設備等を変更・更新し、建物の機能性・快適性・デザイン性を向上させる改修。

建物の基本構造を維持しつつ、エネルギー効率の改善、使い勝手の向上、内装更新などにより、建築当初よりも高い水準の性能・利便性を確保することを目的とする。
修繕

老朽化や損傷が生じた部分を補修し、建物や設備を本来の機能・性能に回復させる工事。

屋根・外壁の補修、設備の故障修理等、建築物の安全性と日常的な使用に支障がない状態を維持することを目的とした計画的・日常的なメンテナンス行為を含む。
本資料におけるPlan C「リノベーション」は、既存図書館建物の機能・性能を抜本的に見直す大規模改修を指す。修繕（原状回復）やリフォーム（部分改修）とは規模・目的が異なる。
